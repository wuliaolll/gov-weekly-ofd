# 政务周报 OFD 生成器 — 需求文档

## 一、项目背景

网站编辑平时在湖北省人民政府门户网站内容管理后台编辑政务周报，现有客户需要政务周报的电子版（OFD 格式），且排版要按一定格式编排。本项目作为轻量工具类需求，采用敏捷开发方式。

## 二、功能需求

### 2.1 核心功能

1. **栏目页自动解析**
   - 用户配置一个栏目页面 URL
   - 自动解析栏目页面中所有周报文档链接

2. **周报详情解析**
   - 解析每篇周报详情页中按日期区分的内容块
   - 识别每个日期下哪些领导有活动动态
   - 获取活动动态的详情链接

3. **文件生成**
   - 访问详情链接获取文章正文内容
   - 同时生成 DOCX 和 OFD 格式文件
   - OFD 中文本须可选中（矢量文本）
   - 排版严格按照公文格式模板编排

4. **定时与手动触发**
   - 每日自动采集生成
   - 也可用户手动触发采集

5. **文件下载**
   - 生成的 DOCX 和 OFD 文件可通过 Web 页面下载

## 三、文件组织结构

```
output/门户网站周报/
├── 2025年/
│   └── 12月8日-12月14日/
│       ├── 李殿勋省长/
│       │   └── 12月8日/
│       │       ├── 李殿勋主持召开省政府常务会议.docx

```

## 四、文稿版面要求

### 4.1 页面设置

| 参数 | 值 |
|------|-----|
| 纸张大小 | A4 (210mm × 297mm) |
| 上边距 | 3.7cm |
| 下边距 | 3.0cm |
| 左边距 | 3.2cm |
| 右边距 | 3.2cm |

### 4.2 字体和字号

| 元素 | 字体 | 字号 |
|------|------|------|
| 标题 | 方正小标宋简体 | 二号 (22pt) |
| 正文 | 仿宋GB2312 | 小二号 (18pt) |
| 数字、英文字母 | Times New Roman | 随所在上下文 |

### 4.3 段落设置

| 参数 | 值 |
|------|-----|
| 对齐格式 | 两端对齐 |
| 特殊格式 | 首行缩进 |
| 缩进量 | 2个字符 |
| 行距 | 固定值 32磅 |
