# 政务周报 OFD 生成器 — 需求文档

## 一、项目背景

网站编辑平时在湖北省人民政府门户网站内容管理后台编辑政务周报，现有客户需要政务周报的电子版（OFD 格式），且排版要按一定格式编排。本项目作为轻量工具类需求，采用敏捷开发方式。

## 二、功能需求

### 2.1 核心功能

1. **栏目页自动解析**
   - 用户配置一个栏目页面 URL（如 `https://www.hubei.gov.cn/hbfb/zwzb/index.shtml`）
   - 自动解析栏目页面中所有周报文档链接

2. **周报详情解析**
   - 解析每篇周报详情页中按日期区分的内容块
   - 识别每个日期下哪些领导有活动动态
   - 获取活动动态的详情链接
   - 没有显著标明领导姓名的区块信息忽略

3. **文件生成**
   - 访问详情链接获取文章正文内容
   - 同时生成 DOCX 和 OFD 格式文件
   - OFD 中文本须可选中（矢量文本）
   - 排版严格按照公文格式模板编排

4. **定时与手动触发**
   - 每日自动采集生成
   - 也可用户手动触发采集和生成每一期的文件

5. **文件下载**
   - 生成的 DOCX 和 OFD 文件可通过 Web 页面下载

### 2.2 产品形式

- 响应式兼容的 Web 页面
- 封装为 Agent Skills 技能，能交给其他 AI 监控、发现、生成

## 三、文件组织结构

```
output/门户网站周报/
├── 2025年/
│   └── 12月8日-12月14日/
│       ├── 李殿勋省长/
│       │   └── 12月8日/
│       │       ├── 李殿勋主持召开省政府常务会议研究加快工业强县建设等工作.docx
│       │       └── 李殿勋主持召开省政府常务会议研究加快工业强县建设等工作.ofd
│       └── 张文兵常务副省长/
│           └── ...
└── 2026年/
    └── 3月30日-4月5日/
        ├── 李殿勋省长/
        │   ├── 3月30日/
        │   │   └── ...
        │   └── 4月1日/
        │       └── ...
        └── 陈平副省长/
            └── ...
```

层级说明：
- 第1层：年份
- 第2层：周报时间范围
- 第3层：领导姓名+职务
- 第4层：活动日期
- 第5层：DOCX 和 OFD 双份文件（以活动标题命名）

## 四、文稿版面要求

### 4.1 页面设置

| 参数 | 值 |
|------|-----|
| 纸张大小 | A4 (210mm × 297mm) |
| 上边距 | 3.7cm |
| 下边距 | 3.0cm |
| 左边距 | 3.2cm |
| 右边距 | 3.2cm |

### 4.2 字体和字号

| 元素 | 字体 | 字号 |
|------|------|------|
| 标题 | 方正小标宋简体 | 二号 (22pt) |
| 正文 | 仿宋GB2312 | 小二号 (18pt) |
| 数字、英文字母 | Times New Roman | 随所在上下文 |

### 4.3 段落设置

| 参数 | 值 |
|------|-----|
| 对齐格式 | 两端对齐 |
| 特殊格式 | 首行缩进 |
| 缩进量 | 2个字符 |
| 行距 | 固定值 32磅 |

### 4.4 其他要求

- 文字颜色：黑色
- 标题：居中编排在第二行，与正文间空一行
- 标题回行：词意完整、排列对称、间距恰当
- 正文：不加粗，不调颜色，不插入图片和视频
- 页码：不设置

## 六、数据源分析

### 6.1 栏目列表页

- URL: `https://www.hubei.gov.cn/hbfb/zwzb/index.shtml`
- 结构: `ul.hbgov-newslist-itemheight-18px > li` 每项包含：
  - `<a href="详情URL">省政府政务周报（日期范围）</a>`
  - `<span>发布时间</span>`

### 6.2 周报详情页

- 按日期分区: 粗体 `**X月X日**` 作为日期分隔
- 领导活动块模式：
  - 模式A: `►►►` + 粗体领导名 + 活动描述 + `详情<<` 链接
  - 模式B: 标题行首直接以领导全名开头 + `详情<<` 链接
- 详情链接指向完整文章页面

### 6.3 文章详情页

- 正文容器: `.bt_content` / `#myText` / `.TRS_Editor`
- 纯文本提取，过滤编辑/审核信息
